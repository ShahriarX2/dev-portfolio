# Shahariar Hossen | Full Stack Developer Portfolio

A high-performance, visually stunning developer portfolio built with Next.js 16, React 19, and modern animation libraries. This project showcases a blend of sharp interfaces, smooth motion, and a clean architectural approach to frontend systems.

## 🚀 Features

- **Immersive UI/UX**: Integrated with `framer-motion`, `GSAP`, and `Lenis` for buttery-smooth scrolling and high-fidelity animations.
- **Interactive Elements**: Features like a `FloatingDock` navigation, `Magnetic` buttons, and `SpotlightCard` components for a premium feel.
- **Dynamic Content**: Project showcases with detailed case studies, including role, outcomes, and technical challenges.
- **Advanced Visuals**: Custom `ShapeGrid` backgrounds with interactive hover effects and a grainy overlay for a modern aesthetic.
- **Responsive Architecture**: Fully responsive design optimized for all device sizes using Tailwind CSS 4.
- **Performance First**: Leveraging Next.js App Router and optimized asset delivery to ensure lightning-fast load times.

## 🛠️ Tech Stack

- **Framework**: [Next.js 16](https://nextjs.org/) (App Router)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/), [GSAP](https://gsap.com/), [Lenis](https://github.com/darkroomengineering/lenis)
- **Backend/Database**: [Supabase](https://supabase.com/)
- **Icons**: [Lucide React](https://lucide.dev/), [Tabler Icons](https://tabler-icons.io/), [React Icons](https://react-icons.github.io/react-icons/)
- **UI Components**: [Shadcn UI](https://ui.shadcn.com/), Radix UI

## 🏁 Getting Started

### Prerequisites

- Node.js 18+
- npm / yarn / pnpm / bun

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/dev-portfolio.git
   cd dev-portfolio
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.
